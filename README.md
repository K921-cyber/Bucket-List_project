# ✅ Bucket List Web Page

A clean and aesthetic **Bucket List** website built using only **HTML** and **CSS**. This project is a fun and simple way to showcase your goals and dreams in a stylish list format — no JavaScript involved!

---

## 🌟 Features

- ✅ Stylish list of personal goals  
- 🎨 Visually appealing layout and design  
- 💡 Built with only HTML and CSS – beginner friendly!

---

## 🛠️ Built With

- HTML5  
- CSS3 (Flexbox, Transitions, Custom Fonts, etc.)
